## Why

**One API, three platforms.** Ship the same component code to web, iOS, and Android. Tamagui handles the renderer; Superstyling gives you Chakra-style ergonomics on top.

**Compile-time extraction.** Style props compile to atomic CSS on the web and flat StyleSheet objects on native — no runtime parser.

**Type-safe theme.** Your palette, spacing, breakpoints, and semantic tokens are typed end to end, so every color prop autocompletes.
