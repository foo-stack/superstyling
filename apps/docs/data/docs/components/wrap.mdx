# Wrap · WrapItem

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

Horizontal flex row that wraps children with consistent `spacing`. `<WrapItem>` is an optional per-child wrapper — signals intent and gives a stable styling slot.

## Import

```tsx
import { Wrap, WrapItem } from "@superstyling/core";
```

## Usage

<ComponentDemo
  code={`<Wrap spacing="$2">
  {tags.map((t) => (
    <WrapItem key={t}>
      <Tag>{t}</Tag>
    </WrapItem>
  ))}
</Wrap>`}
  preview={
    <Wrap spacing="$2">

    </Wrap>
  }
/>
