# Progress

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

Linear progress bar built on `@tamagui/progress`. Pair with `<CircularProgress>` for ring-shaped indicators (SVG on web, styled ring on native).

## Import

```tsx
import { Progress, CircularProgress } from "@superstyling/core";
```

## Linear

```tsx
<Progress value={60} />
<Progress value={30} size="xs" />
<Progress isIndeterminate />
```

## Circular

```tsx
<CircularProgress value={65}>
  <CircularProgress.Label>65%</CircularProgress.Label>
</CircularProgress>
```
