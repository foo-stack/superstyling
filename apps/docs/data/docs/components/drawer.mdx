# Drawer

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

Edge-sliding dialog, ideal for navigation menus, settings panels, and inline detail views. Matches Chakra's `<Drawer>` API surface (placement, size, overlay, focus trap).

## Import

```tsx
import { Drawer } from "@superstyling/core";
```

## Basic

```tsx
const [isOpen, setOpen] = useState(false);

<Drawer isOpen={isOpen} onClose={() => setOpen(false)} placement="right">
  <Drawer.Overlay />
  <Drawer.Content>
    <Drawer.Header>
      <Drawer.Title>Settings</Drawer.Title>
    </Drawer.Header>
    <Drawer.CloseButton />
    <Drawer.Body>{/* … */}</Drawer.Body>
    <Drawer.Footer>
      <Button variant="ghost" onPress={() => setOpen(false)}>
        Cancel
      </Button>
      <Button colorScheme="blue">Save</Button>
    </Drawer.Footer>
  </Drawer.Content>
</Drawer>;
```
