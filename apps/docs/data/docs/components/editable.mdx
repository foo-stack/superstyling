# Editable

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

Inline edit-in-place field. Tap/click `Preview` to swap in an input. `Enter` commits, `Escape` discards, blur commits by default. On native, Android hardware-back while editing discards (fires `onCancel`).

## Import

```tsx
import { Editable } from "@superstyling/core";
```

## Basic

```tsx
<Editable defaultValue="Untitled" onSubmit={(next) => rename(next)}>
  <Editable.Preview />
  <Editable.Input />
</Editable>
```

## Multi-line (textarea)

```tsx
<Editable defaultValue="Draft…">
  <Editable.Preview />
  <Editable.Textarea />
</Editable>
```

## Imperative control

```tsx
function EditableControls() {
  const { isEditing, startEditing, commit, cancel } = useEditableControls();
  return isEditing ? (
    <HStack>
      <Button onPress={commit}>Save</Button>
      <Button onPress={cancel}>Cancel</Button>
    </HStack>
  ) : (
    <Button onPress={startEditing}>Edit</Button>
  );
}
```
