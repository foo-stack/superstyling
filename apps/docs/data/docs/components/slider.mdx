# Slider

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

Continuous-value picker. Built on `@tamagui/slider` — keyboard arrows (←/→), touch drag, and WAI-ARIA slider semantics. Use `<RangeSlider>` for two-thumb range selection.

## Import

```tsx
import { Slider, RangeSlider } from "@superstyling/core";
```
