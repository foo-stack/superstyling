# Slider

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

Continuous-value picker. Built on `@tamagui/slider` — keyboard arrows (←/→), touch drag, and WAI-ARIA slider semantics. Use `<RangeSlider>` for two-thumb range selection.

## Import

```tsx
import { Slider, RangeSlider } from "@superstyling/core";
```

## Single thumb

```tsx
const [value, setValue] = useState([50]);

<Slider value={value} onChange={setValue} min={0} max={100}>
  <Slider.Track>
    <Slider.FilledTrack />
  </Slider.Track>
  <Slider.Thumb index={0} />
</Slider>;
```

## Range (two thumbs)

```tsx
const [range, setRange] = useState([20, 80]);

<RangeSlider value={range} onChange={setRange}>
  <RangeSlider.Track>
    <RangeSlider.FilledTrack />
  </RangeSlider.Track>
  <RangeSlider.Thumb index={0} />
  <RangeSlider.Thumb index={1} />
</RangeSlider>;
```
