# SimpleGrid

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

`<Grid>` with automatic `templateColumns` from either `columns` (fixed count) or `minChildWidth` (responsive auto-fill).

## Import

```tsx
import { SimpleGrid } from "@superstyling/core";
```

## Fixed columns

  code={`<SimpleGrid columns={3} spacing="$3">

</SimpleGrid>`}
