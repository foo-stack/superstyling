# Skeleton

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

Placeholder boxes shown while content loads. Flip `isLoaded` to reveal the real children.

## Import

```tsx
import { Skeleton, SkeletonCircle, SkeletonText } from "@superstyling/core";
```

## Basic

```tsx
<Skeleton height={20} width={200} />
<SkeletonCircle size={40} />
<SkeletonText noOfLines={3} />
```

## Reveal children

```tsx
<Skeleton isLoaded={!loading} height={20}>
  <Text>{data.title}</Text>
</Skeleton>
```
