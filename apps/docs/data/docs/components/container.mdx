# Container

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

Centered max-width wrapper for page content. Default `maxWidth` is `60ch` (readable text-column width); override with any CSS length or token.

## Import

```tsx
import { Container } from "@superstyling/core";
```

## Usage

<ComponentDemo
  code={`<Container maxWidth={720}>
  <Text>Constrained column.</Text>
</Container>`}
  preview={
    <Container maxWidth={720} backgroundColor="$color2" padding="$4">
      <Text>Constrained column.</Text>
    </Container>
  }
/>
