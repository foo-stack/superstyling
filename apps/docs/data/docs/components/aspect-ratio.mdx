# AspectRatio

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

Enforces a `width:height` ratio on its single child. The child stretches to fill the wrapper.

## Import

```tsx
import { AspectRatio } from "@superstyling/core";
```

## Usage

<ComponentDemo
  code={`<AspectRatio ratio={16 / 9}>
  <Box backgroundColor="$color3" />
</AspectRatio>`}
  preview={
    <AspectRatio ratio={16 / 9}>
      <Box backgroundColor="$color3" />
    </AspectRatio>
  }
/>
