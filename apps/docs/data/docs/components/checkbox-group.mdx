# CheckboxGroup

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

Shared context for a set of `<Checkbox>`es. Individual items opt-in by setting a `value` prop; the group tracks selected values and fires `onChange`.

## Import

```tsx
import { CheckboxGroup, Checkbox } from "@superstyling/core";
```

## Controlled

:::tip
See the [Checkbox page](/components/checkbox) for the per-item interactive demo with state wired up.
:::

```tsx
const [selected, setSelected] = useState(["a"]);

<CheckboxGroup value={selected} onChange={setSelected}>
  <Checkbox value="a">Option A</Checkbox>
  <Checkbox value="b">Option B</Checkbox>
  <Checkbox value="c">Option C</Checkbox>
</CheckboxGroup>;
```
