# List · OrderedList · UnorderedList · ListItem · ListIcon

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

Vertical list with optional styleType (disc / decimal / none). `<OrderedList>` emits `<ol>`, `<UnorderedList>` emits `<ul>`.

## Import

```tsx
import { List, OrderedList, UnorderedList, ListItem, ListIcon } from "@superstyling/core";
```

## Unordered list

<ComponentDemo
  code={`<UnorderedList>
  <ListItem>first</ListItem>

  <ListItem>third</ListItem>
</UnorderedList>`}
