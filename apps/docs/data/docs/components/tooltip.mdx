# Tooltip

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

Hover/focus tooltip that wraps a single trigger child. Built on Tamagui's portal + placement flipping.

## Import

```tsx
import { Tooltip } from "@superstyling/core";
```

## Usage

```tsx
<Tooltip label="Save your changes" placement="top">
  <Button colorScheme="blue">Save</Button>
</Tooltip>
```
