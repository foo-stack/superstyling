# Flex

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

Row-direction flex container with shorthand props for `justify`, `align`, `direction`, and `wrap`. Renders as an `XStack` under the hood.

## Import

```tsx
import { Flex } from "@superstyling/core";
```

## Basic usage

<ComponentDemo
  code={`<Flex justify="space-between" align="center" gap="$3">
  <Box>Left</Box>
  <Box>Right</Box>
</Flex>`}
  preview={
    <Flex justify="space-between" align="center" gap="$3">
      <Box>Left</Box>
      <Box>Right</Box>
    </Flex>
  }
/>
