# Popover

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

Floating panel anchored to a trigger. Portaled automatically; handles outside-click dismiss, Escape, and placement flipping.

## Import

```tsx
import { Popover } from "@superstyling/core";
```

## Basic

```tsx
<Popover>
  <Popover.Trigger>
    <Button>Open</Button>
  </Popover.Trigger>
  <Popover.Content>
    <Popover.Arrow />
    <Popover.Header>
      <Text fontWeight="600">Notification</Text>
    </Popover.Header>
    <Popover.Body>
      <Text>Your build just finished. Ship it?</Text>
    </Popover.Body>
    <Popover.Footer>
      <Button size="sm" colorScheme="blue">
        Ship
      </Button>
    </Popover.Footer>
    <Popover.CloseButton />
  </Popover.Content>
</Popover>
```
