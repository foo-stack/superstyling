# PinInput

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

N independent single-character inputs that auto-advance on keypress, backspace-to-previous when empty, and spread a pasted value across all fields. Common for OTP / verification codes.

## Import

```tsx
import { PinInput } from "@superstyling/core";
```

## Basic

```tsx
<PinInput length={4} onComplete={(code) => verify(code)}>
  <PinInput.Field />
  <PinInput.Field />
  <PinInput.Field />
  <PinInput.Field />
</PinInput>
```

## Digits only / alphanumeric

```tsx
<PinInput type="number">…</PinInput>
<PinInput type="alphanumeric">…</PinInput>
```

## Masked

```tsx
<PinInput mask>
  <PinInput.Field />
  <PinInput.Field />
  <PinInput.Field />
  <PinInput.Field />
</PinInput>
```
