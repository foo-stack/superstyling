# ButtonGroup

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

Groups a row of `<Button>`s with shared `size` / `variant` / `colorScheme` / `isDisabled`. `isAttached` produces the segmented-control look.

## Import

```tsx
import { ButtonGroup, Button } from "@superstyling/core";
```

## Spaced group

<ComponentDemo
  code={`<ButtonGroup size="sm" variant="outline" spacing="$2">
  <Button>Save</Button>
  <Button colorScheme="red">Delete</Button>
</ButtonGroup>`}
  preview={
    <ButtonGroup size="sm" variant="outline" spacing="$2">
      <Button>Save</Button>
      <Button colorScheme="red">Delete</Button>
    </ButtonGroup>
  }
/>
