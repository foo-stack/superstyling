# NumberInput

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

Numeric input with up/down stepper buttons and keyboard arrows. Clamps to `[min, max]` on blur, blocks non-numeric keypress, and honors Chakra's `Shift + ↑/↓` = 10× step convention.

## Import

```tsx
import { NumberInput } from "@superstyling/core";
```

## Basic

```tsx
<NumberInput defaultValue={5} min={0} max={10} step={1}>
  <NumberInput.Field />
  <NumberInput.Stepper>
    <NumberInput.IncrementStepper />
    <NumberInput.DecrementStepper />
  </NumberInput.Stepper>
</NumberInput>
```

## Controlled

```tsx
const [value, setValue] = useState(0);

<NumberInput value={value} onChange={(n) => setValue(n)} min={0} max={100}>
  <NumberInput.Field />
  <NumberInput.Stepper>
    <NumberInput.IncrementStepper />
    <NumberInput.DecrementStepper />
  </NumberInput.Stepper>
</NumberInput>;
```

## Keyboard

- `↑` / `↓` — step by `step`
- `Shift + ↑/↓` — step by `step * 10`
- Typing non-numeric characters is blocked at the input
