# Grid · GridItem

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

CSS grid container with Chakra's shorthand prop names. For simple N-equal-column layouts, reach for [SimpleGrid](/components/simple-grid) instead.

## Import

```tsx
import { Grid, GridItem } from "@superstyling/core";
```

## Basic grid

</Grid>`}
