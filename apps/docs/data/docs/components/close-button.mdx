# CloseButton

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

Preset `<IconButton>` with an ✕ icon. Used for dismiss actions on Alerts, Drawers, Modals, Tags.

## Import

```tsx
import { CloseButton } from "@superstyling/core";
```

## Usage

<ComponentDemo
  code={`<CloseButton aria-label="Close banner" onPress={() => setDismissed(true)} />`}
  preview={<CloseButton aria-label="Close demo" />}
/>
