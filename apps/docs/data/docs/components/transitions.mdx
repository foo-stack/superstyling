# Fade · ScaleFade · Slide · Collapse

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

Animation wrappers matching Chakra's `<Fade>`, `<ScaleFade>`, `<Slide>`, `<Collapse>`. All share the same API: pass `in={true|false}` to mount/unmount with an enter/exit animation.

## Import

```tsx
import { Fade, ScaleFade, Slide, Collapse } from "@superstyling/core";
```

## Fade

```tsx
<Fade in={isOpen}>
  <Text>Fades in and out</Text>
</Fade>
```

## ScaleFade

```tsx
<ScaleFade in={isOpen} initialScale={0.9}>
  <Text>Scales + fades</Text>
</ScaleFade>
```

## Slide

```tsx
<Slide in={isOpen} direction="bottom" offset={60}>
  <Text>Slides in from below</Text>
</Slide>
```

Directions: `"top" | "right" | "bottom" | "left"`.

## Collapse

```tsx
<Collapse in={isOpen} startingHeight={0}>
  <Text>Collapses vertically</Text>
</Collapse>
```
