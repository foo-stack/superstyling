# Table

<StatusRow tier="1" platforms="Cross-platform" since="v0.2.0" />

Tabular data with platform-appropriate rendering. On web, renders as real `<table>` / `<thead>` / `<tbody>` / `<tr>` / `<th>` / `<td>` for row/columnheader announcements. On native, same compound API renders as flex boxes — wrap long lists in `FlatList` yourself to keep the primitive thin.

## Import

```tsx
import { Table, TableContainer } from "@superstyling/core";
```

## Basic

```tsx
<TableContainer>
  <Table>
    <Table.Caption>Users</Table.Caption>
    <Table.Thead>
      <Table.Tr>
        <Table.Th>Name</Table.Th>
        <Table.Th>Email</Table.Th>
        <Table.Th isNumeric>Age</Table.Th>
      </Table.Tr>
    </Table.Thead>
    <Table.Tbody>
      {users.map((u) => (
        <Table.Tr key={u.id}>
          <Table.Td>{u.name}</Table.Td>
          <Table.Td>{u.email}</Table.Td>
          <Table.Td isNumeric>{u.age}</Table.Td>
        </Table.Tr>
      ))}
    </Table.Tbody>
  </Table>
</TableContainer>
```
