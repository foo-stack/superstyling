# Heading

<StatusRow tier="1" platforms="Cross-platform" since="v0.1.0" />

Semantic heading primitive. On web the `level` prop maps to the matching `h1`–`h6` element; on native it always renders as `Text` with the visual size applied so screen readers announce the correct hierarchy.

## Import

```tsx
import { Heading } from "@superstyling/core";
```

## Accessibility

:::tip
Keep headings in document order and don't skip levels. Superstyling preserves the semantic element on web and sets the correct `accessibilityRole="header"` plus `aria-level` on native, so both platforms surface the same heading hierarchy to assistive tech.
:::
