
# Text

<StatusRow tier="1" platforms="Cross-platform" since="v0.1.0" />

The primary inline-text primitive. Maps to a platform-appropriate text element (a plain `<span>` on web, RN `<Text>` on native) and accepts every typography token and style prop the system knows.
