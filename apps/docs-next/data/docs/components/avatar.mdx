
# Avatar

<StatusRow tier="1" platforms="Cross-platform" since="v0.1.0" />

User-image primitive with a fallback slot. Mirrors Tamagui's `Avatar` compound structure — `Avatar.Image` + `Avatar.Fallback` — so the fallback renders whenever the image fails to load.

## Import

```tsx
```

## Basic usage

<ComponentDemo
  code={`<Avatar circular size="$6">
  <Avatar.Image src="https://i.pravatar.cc/150?img=1" />
  <Avatar.Fallback backgroundColor="$primaryMuted" />
</Avatar>`}
  preview={
    <Avatar circular size="$6">
      <Avatar.Image src="https://i.pravatar.cc/150?img=1" />
      <Avatar.Fallback backgroundColor="$primaryMuted" />
    </Avatar>
  }
/>
