
# Link

<StatusRow tier="1" platforms="Cross-platform" since="v0.1.0" />

Anchor primitive. Renders a real `<a>` on web so the browser handles keyboard, middle-click, and copy-link. On native it falls back to a pressable that invokes the Linking API.

## Accessibility

:::tip
External links should have visible text that makes the destination obvious (avoid "click here"). Link's default focus ring comes from the underlying platform element; override via `_focus` if you need a custom state.
:::
