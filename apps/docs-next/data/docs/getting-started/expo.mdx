# Getting Started — Expo

Zero to `<Button>` on a fresh Expo SDK 54+ project with Expo Router.

## 1. Install

```bash
yarn add @superstyling/core @superstyling/expo @superstyling/icons tamagui
yarn add react-native-gesture-handler react-native-safe-area-context react-native-teleport react-native-worklets
```

The native peers are optional — each enables a feature (gesture-handler powers Sheet/Modal drag, teleport powers the native Portal host). Install only what you use.

## 2. Create `tamagui.config.ts`

```tsx
// tamagui.config.ts
export const system = createSystem({});
export default system.config;
declare module "tamagui" {
  interface TamaguiCustomConfig extends typeof system.config {}
}
```

## 3. Wire the Babel plugin

```js
// babel.config.js
const { babelPreset } = require("@superstyling/expo/babel-plugin");

module.exports = function (api) {
  api.cache(true);
  return {
    presets: ["babel-preset-expo"],
    plugins: babelPreset({ config: "./tamagui.config.ts" }),
  };
};
```

## 4. Wire Metro

```js
// metro.config.js
const { getDefaultConfig } = require("expo/metro-config");
const { withSuperStylingMetro } = require("@superstyling/expo/metro-config");

const config = getDefaultConfig(__dirname, { isCSSEnabled: true });
module.exports = withSuperStylingMetro(config, {
  projectRoot: __dirname,
  config: "./tamagui.config.ts",
});
```

## 5. Register the native setup

```tsx
// app/_layout.tsx (Expo Router)
import "@superstyling/expo/setup";
import { Slot } from "expo-router";

export default function RootLayout() {
  return (
    <SuperStylingProvider>
      <Slot />
    </SuperStylingProvider>
  );
}
```

The `import "@superstyling/expo/setup"` must be the **first** import — it wires Tamagui's native portal, gesture handler, worklets, safe-area, and keyboard controller in the required order.

## 6. Register the config plugin (optional)

```jsonc
// app.json
{
  "expo": {
    "plugins": ["@superstyling/expo"],
  },
}
```

## 7. Render a Button

```tsx
// app/index.tsx

export default function Home() {
  return <Button colorScheme="blue">Hello from Superstyling</Button>;
}
```

## Troubleshooting

:::warning[Portal not working on native]
Make sure `@superstyling/expo/setup` is the first import in your entry file and that `react-native-teleport` is installed. Without both, Tamagui silently falls back to a JS portal that drops React context.
:::

:::info[Optional integrations]
Swap `@superstyling/expo/setup` for `/setup-all` to enable `expo-linear-gradient`, `burnt` toasts, and `zeego` menus when those peers are installed.
:::
