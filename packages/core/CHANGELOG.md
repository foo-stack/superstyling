# @superstyling/core

## 0.1.0

### Minor Changes

- 50c58d5: Initial v0.1.0 release — a Chakra-UI-shaped React component library built on Tamagui v2. One API for web, iOS, and Android; compile-time atomic CSS on web and flat StyleSheet on native; type-safe Chakra-style theme via `createSystem`.

  **Components shipped:** `Box`, `Stack`/`HStack`/`VStack`, `Text`, `Heading`, `Divider`, `Spinner`, `Badge`, `Avatar`, `Button`, `IconButton`, `Link`, `Alert`, `Modal`, `FormControl`, `Input`, `Textarea`, `Checkbox`, `Radio`/`RadioGroup`, `Switch`, `Select`.

  **Integrations:** `@superstyling/next` (App Router + Pages Router, `withSuperStyling`, `ColorModeScript`), `@superstyling/expo` (`/setup`, `/setup-all`, `/babel-plugin`, `/metro-config`, `/app-plugin`), `@superstyling/vite` (`superstylingVitePlugin`, `superstylingAliases`, `colorModeScriptSnippet`).

  **Icons:** 21 icons in `@superstyling/icons` (chevrons, arrows, close, check, plus, minus, search, edit, trash, sun, moon, etc.) with platform-split SVG + custom icon factory.
